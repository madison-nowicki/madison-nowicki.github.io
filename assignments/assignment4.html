Challenge 5: Accessibility Statement

Accessibility is important to consider when coding anything. Individuals with visual impairments rely on supportive technology, like screen reading softwares, to assist them in navigating webpages. For Challenge 2: Build a Chessboard, for example, a screen reader will detect the code for the table and trigger commands to bring the user through the table, both horizontally and vertically. However, if the code isn't well-structured and uniform, the screen reader will deliver unordered information to the user, leaving the user confused.
